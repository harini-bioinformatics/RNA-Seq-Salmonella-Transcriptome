# Transcriptomic Analysis of Salmonella enterica

## Overview
This project focuses on RNA-Seq based transcriptome analysis to identify unique and strain-specific transcripts.

## Tools Used
- FastQC
- Trimmomatic
- Trinity
- BLAST
- PANTHER (GO Analysis)
- Venny
- R (PCA, Heatmap, Volcano Plot)

## Workflow
1. Data retrieval from NCBI SRA
2. Quality control using FastQC
3. Read trimming using Trimmomatic
4. De novo assembly using Trinity
5. Sequence annotation using BLAST
6. Functional analysis using GO
7. Comparative analysis using Venny
8. Visualization using PCA, Heatmap, Volcano plots

## Results
- Identified unique transcripts
- Functional annotation of genes
- Insights into virulence and resistance
